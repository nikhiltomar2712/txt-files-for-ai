## Real-world next step (what to implement next)
To turn this into a *real* LLM app, you typically need:
- A real tokenizer (BPE/SentencePiece)
- A model backend (transformers / inference server)
- Prompt formatting + system/user roles
- Streaming output + conversation memory
- Guardrails (safety, content filters)

(These files are for structure learning; they do not depend on large ML models.)
