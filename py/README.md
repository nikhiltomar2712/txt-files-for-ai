# Python examples (AI + toy LLM)

## 1) `llm_simple_chat.py`
A tiny *toy* LLM-like generator that demonstrates the pipeline:
- prompt -> tokens
- context window
- generation loop

It does **not** implement a real neural network, but it’s useful for learning structure.

Run:

```bash
python3 llm_simple_chat.py
```

Then type questions like:
- `How does AI training work?`
- `How can I build an LLM?`
